# Spartan Marketplace Frontend (Vite + React + TypeScript + Tailwind)

Install dependencies and run the dev server:

```bash
cd frontend
npm install
npm run dev
```

The app will run at `http://localhost:3000` by default.

Notes:
- Tailwind is configured in `tailwind.config.cjs` and styles in `src/styles/tailwind.css`.
- Pages are in `src/pages` (`Index`, `Placeholder`, `NotFound`).
